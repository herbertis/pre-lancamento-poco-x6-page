# 🚀 POCO X6 - Pre-Launch Landing Page

[Preview do Projeto](https://herbertis.github.io/pre-lancamento-poco-x6/)

## 💻 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Landing Page de pré-lançamento para o POCO X6**, criada para gerar expectativa e engajamento antes da chegada oficial do produto ao mercado brasileiro.

A página foi desenvolvida para ser utilizada nos **canais oficiais da Xiaomi Brasil (Mi Brasil)**, seguindo os padrões visuais e estruturais das campanhas de lançamento da marca.

O objetivo foi criar uma experiência visual impactante e imersiva, destacando o conceito do produto e preparando os usuários para o lançamento oficial através de **uma comunicação visual forte, seções promocionais e narrativa focada em tecnologia e inovação**.

A landing page foi construída com foco em **performance, responsividade e experiência do usuário**, simulando páginas utilizadas em campanhas de pré-lançamento de grandes marcas de tecnologia.

---

## 🚀 Principais Funcionalidades (Features)

* **Hero Section de Pré-Lançamento:** Destaque visual do produto com comunicação focada em gerar expectativa para o lançamento.
* **Storytelling de Produto:** Estrutura narrativa que apresenta os principais diferenciais da linha POCO.
* **Layout Promocional de Campanha:** Interface inspirada em páginas de lançamento utilizadas por grandes marcas de tecnologia.
* **Design Responsivo:** Interface adaptável para **Mobile, Tablet e Desktop**.
* **Seções Visuais de Destaque:** Blocos visuais que reforçam inovação, performance e design do dispositivo.
* **Performance Front-end:** Estrutura leve e otimizada para carregamento rápido.

---

## 🛠️ Tecnologias Utilizadas

Neste projeto foi adotada uma abordagem **clean e performática**, priorizando controle da interface e carregamento rápido da página.

* **HTML5:** Estrutura semântica e organização do conteúdo.
* **CSS3:** Flexbox, CSS Grid, responsividade, animações e variáveis CSS.
* **JavaScript (Vanilla):** Manipulação de DOM e interações da interface.
* **Responsive Design:** Media Queries para adaptação em múltiplos dispositivos.

---

## 🎨 Abordagem de Design (UI/UX)

Como profissional que une **Design e Desenvolvimento Front-end**, o foco desta landing page foi criar uma experiência visual capaz de gerar impacto e curiosidade antes do lançamento do produto.

Principais princípios aplicados:

1. **Hierarquia Visual:** Destaque estratégico das seções promocionais.
2. **Design Orientado à Campanha:** Estrutura pensada para gerar expectativa no usuário.
3. **Experiência Mobile First:** Interface pensada inicialmente para dispositivos móveis.
4. **Estética Tecnológica:** Layout alinhado com a identidade visual da marca POCO.

---

## 📈 O Impacto (Business Value)

Landing pages de pré-lançamento são fundamentais para **estratégias de marketing digital e lançamento de produtos tecnológicos**.

Principais benefícios desta abordagem:

1. **Geração de Expectativa:** Criação de antecipação para o lançamento oficial do produto.
2. **Engajamento do Usuário:** Estrutura visual que estimula curiosidade e exploração da página.
3. **Fortalecimento da Marca:** Comunicação alinhada com a identidade da POCO.
4. **Preparação para Campanhas de Lançamento:** Base para divulgação em campanhas digitais e redes sociais.
